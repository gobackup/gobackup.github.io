# GoBackup Website

https://gobackup.github.io

This documentation site is built with [Mintlify](https://mintlify.com).

## Development

```bash
npm install
# or
bun install
```

### Start Dev server

```bash
npm run dev
# or
bun run dev
```

This will start the Mintlify dev server at `http://localhost:3000`

## Build

```bash
npm run build
# or
bun run build
```

## Project Structure

- `docs/` - Documentation content files (MDX format)
- `mint.json` - Mintlify configuration file
- `public/` - Static assets (images, logos, etc.)
